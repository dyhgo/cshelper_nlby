cshelper功能介绍
cshelper共分为三个部分，移动端APP，数据存储系统，交流区。
一、移动端

  1. 网页浏览
    • 迎合用户需求，设置多种浏览模式。
    • 聚合专业相关的各类网站，提供便捷入口。
    • 集成各类网页工具，供用户快捷使用。
  2. 书签历史
    • 书签分类自定义，历史记录按日期存储，方便用户整理。
    • 浏览器界面提供当前界面完整历史记录，可进行直接跳转访问。
  3. 笔记编辑
    • 可进行纯文本、图像、录音等多种形式笔记的编辑存储。
    • 笔记浏览界面提供搜索等功能。
  4. 数据同步
    • 针对笔记、书签等关键数据，用户可通过上传选项一键上传，进行后台备份和管理。
  5. 交流社区
    • 用户可将自己的经验心得在交流区进行分享，学习方面的问题也可以发布讨论。
  6. 个人题库
    • 收集在移动端查看编程题目的链接，编辑程序的代码等数据，按个人需求进行存储。

二、数据存储系统

  1. 日志记录
    • 在Web端提供富文本编辑器进行日志记录，可使用Markdown进行编辑，记录个人学习心得、问题解决思路等。
    • 列表形式展示用户全部日志，提供新建、删除、更新、搜索等功能。
  2. 移动端数据
    列表形式展示由移动端上传的书签、笔记以及题目等数据，进行查阅和整理。

三、交流区
交流区分为管理端和论坛端。
管理端

  1. 用户管理
    • 浏览系统用户信息（基本信息和登录注册统计）。
    • 修改用户状态（拉黑）。
  2. 帖子管理
    • 浏览帖子信息。
    • 修改帖子状态（隐藏、置顶等）。
  3. 版块管理
    • 版块：新建版块。
    • 版块：编辑已有版块信息。
    • 版块：删除版块。
    • 分类：新建分类。
    • 分类：删除分类。

论坛端

  1. 用户功能
    • 用户注册。
    • 用户登录（以注册邮箱为用户名）。
    • 用户设置，完善身份信息。
    • 用户主页，查看自己的发布的帖子等信息。
    • 私信，提供两用户间的不公开直接交流。
  2. 帖子浏览
    • 帖子浏览，可基于版块、分类进行筛选。
    • 帖子搜索。
    • 帖子评论，直接针对帖子内容进行评论。
    • 帖子回复，针对评论进行回复。
    • 帖子收藏，可自定义收藏夹。
  3. 帖子编辑
    • 发布帖子，选定版块、分类进行发布。
    • 帖子修改，可对已发布帖子进行修改。
  4. 版块管理（版主用户特有）
    • 浏览帖子信息。
    • 修改帖子状态（隐藏、置顶等）。
    • 分类：新建分类。
    • 分类：删除分类。
